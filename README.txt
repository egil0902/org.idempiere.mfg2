org.idempiere.mfg2
=============

Description
-----------
iDempiere Manufacturing level 2 plugin is based on work done by Eevolution in libero manufactuing in Adempiere project.
Red1 has converted Adempiere module into iDempiere plugin and later Peter Shepetko has made it compitable with iDempiere 7.1 and 8.2.
The goal of this project is to enhance this plugin, fix issue and restructure.
